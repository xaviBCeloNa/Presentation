Miles Per Gallon

  1. The application allows you to fit a linear model to the variable mpg from the data set mtcars
  2. You can explain the mpg with a some possible explicative variables: Displacement, Gross Horsepower, Weight, Rear axle ratio and V/S
  3. A scatterplot represent both variables and different check box are available: Show Confidence Interva, Smooth the Data and Show the linear model estimated.
